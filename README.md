🌱 Habit Pulse

A modern, minimal habit tracking application designed to help users build consistency, track progress, and improve daily routines.

Habit Pulse makes it simple to create habits, monitor streaks, and gain insights into your personal growth journey.

✨ Features
✅ Create and manage daily habits
🔥 Track streaks and consistency
📊 Visualize progress over time
🧠 Simple and intuitive UI
🔔 Reminders & notifications (if implemented)
🌙 Clean and minimal design
